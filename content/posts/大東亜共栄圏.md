# 大東亜共栄圏

> 「アジアはアジア人のもの」——この言葉が掲げられた時、その裏にあったのは解放の理念か、それとも新たな支配の論理か。

---

## 1. はじめに

大東亜共栄圏（だいとうあきょうえいけん、英: Greater East Asia Co-Prosperity Sphere）は、**1940年代の日本帝国が掲げた地政学的・イデオロギー的構想**である。

欧米列強によるアジア支配からの「解放」を名目に掲げつつ、実態としては日本を盟主とするアジア広域支配圏の形成を目指したものであり、その評価は今なお論争的である。

この記事では以下の観点から大東亜共栄圏を整理する。

- 概念の形成過程
- 地理的範囲と参加国・地域
- 宣伝と実態のギャップ
- 各地域における影響
- 戦後の評価と現代的意義

---

## 2. 概念の形成

### 2.1 背景

20世紀前半のアジアは、イギリス・フランス・オランダ・アメリカによる植民地支配が広がっていた。

| 地域 | 宗主国 | 支配開始（おおよそ） |
|------|--------|----------------------|
| インド | イギリス | 1858年 |
| インドシナ | フランス | 1887年 |
| オランダ領東インド | オランダ | 17世紀〜 |
| フィリピン | アメリカ | 1898年 |
| マラヤ | イギリス | 1867年〜 |

日本は日清戦争（1894-95）・日露戦争（1904-05）を経て列強の一角に入ったが、**1930年代の世界恐慌と資源不足**が膨張主義的政策を加速させた。

### 2.2 「共栄圏」という言葉の登場

1940年8月、松岡洋右外相が正式に「大東亜共栄圏」という用語を使用したのが公式の初出とされる。

この構想は主に以下の要素から成っていた。

1. **反欧米・反植民地主義**のイデオロギー
2. 日本を核とした**ブロック経済圏**の形成
3. 「八紘一宇（はっこういちう）」思想との接続
4. アジア各民族の「共存共栄」という建前

```
八紘一宇：
「世界を一つの家とする」という日本の国家神道的理念。
大東亜共栄圏のイデオロギー的基盤の一つとされた。
```

## 3. 地理的範囲

大東亜共栄圏として想定・実効支配された地域は以下のとおりである。

### 3.1 核心地域

- **満洲国**（1932年建国、傀儡国家）
- **中華民国（汪兆銘政権）**（1940年〜、南京）
- **朝鮮半島**（1910年より併合）
- **台湾**（1895年より領有）

### 3.2 占領地域

太平洋戦争開戦（1941年12月）後、急速に拡大した。

```
1941年12月 - マレー半島上陸、香港占領
1942年1月  - マニラ占領
1942年2月  - シンガポール陥落（「昭南島」と改称）
1942年3月  - ジャワ島占領
1942年5月  - ビルマ（ミャンマー）占領
```

### 3.3 最大版図のイメージ

```
北: アリューシャン列島（一部）
西: ビルマ・インド国境付近
南: ニューギニア・ソロモン諸島
東: ウェーク島・ギルバート諸島
```

> **NOTE:** この版図は1942年中頃の最大拡大期のもの。ミッドウェー海戦（1942年6月）以降、急速に縮小した。

## 4. 建前と実態

### 4.1 公式の理念

日本政府・軍部が掲げた理念は以下のようなものだった。

- ~~欧米の植民地支配~~ からアジアを解放する
- 各民族が**自存自衛**できる体制を作る
- 経済的に「共存共栄」する圏域を構築する

1943年11月には**大東亜会議**が東京で開催され、日本・満洲国・中華民国（汪政権）・タイ・ビルマ・フィリピン・インド（自由インド仮政府）の代表が参加した。

### 4.2 実態との乖離

しかし現実は理念と大きく乖離していた。

| 項目 | 建前 | 実態 |
|------|------|------|
| 独立 | 各民族の自決を尊重 | 傀儡政権の設置 |
| 経済 | 共存共栄 | 資源の日本への一方的収奪 |
| 文化 | 各文化の尊重 | 皇民化政策・日本語強制 |
| 労働 | 協力関係 | 強制労働・慰安婦問題 |
| 食料 | 相互供給 | 飢餓の発生（ベンガル等） |

> ⚠️ **歴史的論点**
>
> 「解放の側面があった」という評価と「新たな植民地支配だった」という評価は、現在も歴史家の間で議論が続いている。
> 一方的な結論を出すことは難しく、地域・時期・文脈によって状況は大きく異なる。

## 5. 各地域における影響

### 5.1 東南アジア

#### フィリピン

日本占領下でホセ・ラウレルを大統領とする**フィリピン共和国**（1943年）が独立を宣言。しかし実質的な自治は極めて限定的であった。

> 「日本軍はスペインやアメリカよりも残酷だった」——フィリピン独立運動家の証言（複数）

#### インドネシア

オランダ植民地支配からの解放という文脈で、一部では日本を歓迎する動きもあった。スカルノはこの時期に日本と協力関係を持ちつつも、独立の機会を模索した。

**1945年8月17日**、日本の敗戦直後にスカルノ・ハッタがインドネシア独立を宣言。大東亜共栄圏の崩壊が、皮肉にも独立のきっかけとなった。

#### ビルマ（現ミャンマー）

- 1943年8月に「独立」宣言（バー・モウ政権）
- 実態は日本軍政下
- **泰緬鉄道建設**において多数の連合軍捕虜・アジア人労働者が死亡

### 5.2 朝鮮・台湾

朝鮮・台湾は「内地延長」として日本本土に近い扱いとされ、**皇民化政策**が強力に推進された。

- 創氏改名（朝鮮人に日本式氏名の使用を強制）
- 神社参拝の強制
- 朝鮮語教育の制限
- 徴兵・徴用工・慰安婦問題

> この時期の問題は現在も**日韓・日中関係の外交的懸案**として残っている。

### 5.3 中国

汪兆銘政権（南京）は日本の支援で樹立されたが、蒋介石の重慶国民政府・毛沢東の共産党と三つ巴の状態が続いた。

```
重慶国民政府（蒋介石）← 米英の支援
南京国民政府（汪兆銘）← 日本の支援
延安共産党（毛沢東）  ← 独自路線
```

## 6. 崩壊

### 6.1 転換点

| 年月 | 出来事 |
|------|--------|
| 1942年6月 | ミッドウェー海戦で日本海軍が大敗 |
| 1943年2月 | ガダルカナル島撤退 |
| 1944年7月 | サイパン陥落、東条内閣総辞職 |
| 1945年3月 | 東京大空襲 |
| 1945年8月6日 | 広島に原爆投下 |
| 1945年8月8日 | ソ連対日参戦 |
| 1945年8月9日 | 長崎に原爆投下 |
| 1945年8月15日 | 終戦の詔書（玉音放送） |
| 1945年9月2日 | 降伏文書調印 |

### 6.2 崩壊の要因

大東亜共栄圏が崩壊した要因は複合的であった。

1. **軍事的敗北** — 制海権・制空権の喪失
2. **経済的破綻** — 資源輸送路の遮断
3. **現地の抵抗運動** — フィリピン、ビルマ、中国など各地での抵抗
4. **理念と実態の乖離** — 「解放」の名目が現地で信頼されなかった
5. **原爆投下とソ連参戦** — 最終的な終戦への圧力

## 7. 戦後の評価

### 7.1 「解放論」

一部の論者（主に日本の保守系歴史家・アジアの一部の民族主義者）は、大東亜共栄圏がアジアの脱植民地化を促進したと評価する。

- インドネシア、ビルマ、インドなど戦後独立が相次いだこと
- 日本軍が欧米軍を破ったことが「白人神話」を崩したこと
- 大東亜会議が人種差別撤廃を議題としたこと

### 7.2 「侵略論」

多くの歴史家・被害国の立場からは、大東亜共栄圏は本質的に帝国主義的侵略であったと評価される。

- 強制労働・慰安婦・民間人虐殺などの人権侵害
- 経済的収奪による現地経済の破壊
- 建前としての「独立」と実態としての軍政支配の乖離

### 7.3 現代的評価の難しさ

~~単純な善悪二元論では語れない~~ のが実情である。

- 地域によって経験が大きく異なる
- 加害と被害が複雑に絡み合っている
- 「戦後アジア独立」への影響の評価が分かれる

## 9. まとめ

大東亜共栄圏は、以下の3点で総括できる。

> **理念と現実の乖離が最大の問題であった。**

1. **理念としての反植民地主義** は当時のアジアで一定の共鳴を呼んだ
2. **実態としての軍事支配・収奪** は理念を裏切り、各地に深刻な被害をもたらした
3. **歴史的遺産** は複雑であり、戦後アジアの独立運動に影響を与えた面と、戦争犯罪・植民地支配の遺制という面の両方を持つ
